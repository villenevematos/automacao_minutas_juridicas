# Automação de Minutas Jurídicas

Sistema desenvolvido para automatizar a geração de documentos jurídicos complexos a partir de templates dinâmicos. O script processa dados estruturados e preenche minutas contratuais automaticamente, reduzindo o erro humano e o tempo de elaboração em até 80%.

Tecnologias:
- Python
- python-docx (Manipulação de Word)
- Pandas (Gestão de dados)

Funcionalidades: 
- Leitura de dados via Excel/JSON.
- Mapeamento inteligente de campos variáveis.
- Geração de múltiplos arquivos formatados em segundos.
